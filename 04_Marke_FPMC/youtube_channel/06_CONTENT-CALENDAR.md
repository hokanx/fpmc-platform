## Weekly production rhythm (one sitting each, ~2–3 h)

1. **Mon** — write/finalize next episode script + prompts (free)
2. **Tue** — produce clips + voice (credits; calibrated batch)
3. **Wed** — QC, fix blocks, assemble, thumbnails, metadata
4. **Thu** — upload + schedule for Saturday; produce Shorts from last episode
5. **Sat** — publish; note first-48h CTR/retention in this file
6. Review analytics after EP03: double down on the best-performing story type.

## KPIs to watch (YouTube Studio)

- CTR ≥ 4% (else: thumbnail/title iteration — A/B is on)
- Avg. view duration ≥ 50% (else: tighten hooks in blocks 1–3)
- Path to monetization: 1,000 subs + 4,000 watch hours (public + >1 min videos)
